JusticeBox — Legal AI Assistant
RAG-powered document Q&A chatbot for legal queries
Python LangChain RAG ChromaDB Streamlit Groq LLM

JusticeBox is a full-stack legal AI assistant that lets users upload legal documents (PDF, DOCX, TXT) and chat with an AI that answers questions grounded in those documents — not hallucinated facts.

How it works: Documents are chunked and converted into vector embeddings using FastEmbed, then stored in a ChromaDB vector database. When a user asks a question, the system retrieves the most relevant document chunks and feeds them into Groq's LLM using a LangChain RAG chain, producing grounded, source-cited answers.

  • Secure user authentication with SHA-256 password hashing and persistent JSON storage
  • Custom dark-navy themed UI with looping video background and animated navigation bar
  • Chat history tracking, session statistics, and a settings panel for data management
  • Source attribution — shows which uploaded document(s) the answer was drawn from

Water Pollutants Predictor
ML-powered web app to forecast chemical pollutant levels
Python Scikit-learn Pandas NumPy Streamlit Joblib

A machine learning web application that predicts water pollutant concentrations at monitoring stations based on year and station ID. Built to support environmental awareness and early warning for water quality degradation.

How it works: A trained regression model (serialized via Joblib) takes the year and one-hot encoded station ID as input and outputs predicted levels for six key chemical pollutants. The Streamlit interface provides an intuitive form for real-time prediction.

  • Predicts 6 pollutants: O₂, NO₃, NO₂, SO₄, PO₄, and Cl simultaneously
  • Handles unseen station IDs gracefully by aligning feature columns with training schema
  • Model and column schema persisted with Joblib for fast, consistent inference
